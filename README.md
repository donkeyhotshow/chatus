# ChatUs

Приватный чат без регистрации — общайтесь, рисуйте и играйте вместе.

## 🌐 Публичный линк

**Production:** https://chatus-omega.vercel.app/

## 🚀 Возможности

- 💬 **Чат** — мгновенные сообщения в реальном времени
- 🎨 **Холст** — рисуйте вместе на общем холсте
- 🎮 **Игры** — 7 мини-игр для компании

## 🛠️ Технологии

- Next.js 14
- TypeScript
- Firebase (Firestore, Auth, Storage)
- Tailwind CSS
- Vercel (хостинг)

## 📦 Установка

```bash
npm install
npm run dev
```

## 🔧 Переменные окружения

Создайте `.env.local` на основе `.env.example`
