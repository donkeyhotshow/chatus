# Инстпо тестированию чата

## Локальное тестирование

1. **Запуск приложения:**
   ```bash
   npm run dev
   ```
   Приложение будет доступно на http://localhost:3000

2. **Тестирование основного функционала:**
   - Откройте http://localhost:3000
   - Введите ник и код комнаты (например: TEST)
   - Нажмите "Войти в чат"

3. **Тестирование реального времени:**
   - Откройте http://localhost:3000/chat-test
   - Откройте эту же страницу в другом браузере/вкладке
   - Отправьте сообщения и проверьте, что они появляются в реальном времени

## Продакшн тестирование

1. **URL приложения:**
   - Основной URL: https://chatus-app.vercel.app (или аналогичный)
   - Тестовая страница: https://chatus-app.vercel.app/chat-test

2. **Тестирование между пользователями:**
   - Откройте приложение на разных устройствах
   - Войдите в одну комнату с разными никами
   - Проверьте обмен сообщениями в реальном времени
   - Проверьте индикаторы онлайн статуса

3. **Что проверить:**
   - ✅ Загрузка приложения без зависания
   - ✅ Вход в комнату чата
   - ✅ Отправка и получение сообщений
   - ✅ Отображение сообщений в реальном времени
   - ✅ Индикаторы печати (typing indicators)
   - ✅ Онлайн статус пользователей
   - ✅ Работа на мобильных устройствах

## Отладка

Если что-то не работает:

1. **Откройте консоль браузера (F12)**
2. **Проверьте страницу отладки:** /debug
3. **Проверьте логи Firebase в консоли**

## Firebase конфигурация

Убедитесь, что все переменные окружения настроены:
- NEXT_PUBLIC_FIREBASE_API_KEY
- NEXT_PUBLIC_FIREBASE_PROJECT_ID
- NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN
- NEXT_PUBLIC_FIREBASE_DATABASE_URL
- И другие Firebase переменные
