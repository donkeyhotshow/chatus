# Папка для стикеров

Эта папка предназначена для хранения файлов стикеров, которые используются в чате.

## Как добавить новые стикеры?

1.  **Поместите файлы** в эту директорию (`public/jjaba/png`).
2.  **Поддерживаемые форматы:** `.png` и `.webp`.
3.  **Всё!** Приложение автоматически обнаружит и отобразит новые стикеры в меню выбора стикеров.

Перезапускать сервер или вносить изменения в код не нужно.
